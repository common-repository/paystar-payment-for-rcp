درگاه پرداخت پی استار برای افزونه Restrict Content Pro وردپرس
برای درخواست پذیرندگی و دریافت اطلاعات ترمینال از لینک زیر اقدام نمایید :
https://my.paystar.ir/

== Description ==

with this plugin, you can easily add paystar.ir gateway to "Restrict Content Pro" plugin
با این پلاگین میتونید درگاه پرداخت پی استار رو به سیستم پرداخت افزونه "Restrict Content Pro" اضافه کنید


== Installation ==
 
just enter paystar.ir terminal (API-Key)
راه اندازی و پیکربندی افزونه نیاز به تنظیمات خاصی نیست. فقط کافیه کد ترمینال پذیرندگی خودتون رو در تنظیمات افزونه وارد کنید
برای ورود به صفحه تنظیمات افزونه از منوی یا عضویت وی آی پی ، زیرمنوی تنظیمات ، به سربرگ پرداخت ها رفته و با زدن تیک درگاه پی استار در بخش فعال کردن درگاه بانکی اون رو فعال کنید
پایین همین صفحه میتونید ترمینال رو وارد کنید
